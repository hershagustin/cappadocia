# Cappadocia Project

## Overview
Cappadocia is a responsive, mobile-first website built using **DaisyUI**, a Tailwind CSS component library. This project showcases modern UI components, responsive layouts, and structured design systems, combined with subtle CSS animations to enhance user interaction.

## Features
- Fully responsive design optimized for mobile, tablet, and desktop screens
- Interactive UI elements using **CSS animations**
- Three fully developed pages demonstrating practical layout and component usage
- Clean and organized structure for maintainable and scalable code

## Technologies Used
- HTML
- DaisyUI (Tailwind CSS component library)
- CSS Animations
